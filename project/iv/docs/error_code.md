# IV 诊断错误码定义

本文档定义运行时 `rulemap_final_summary.csv` 中使用的表象编码、根因编码和运维建议编码。

## 表象异常 (1001-1010)

| 编码 | 字段 | 描述 |
|------|------|------|
| 1001 | dIsc | 短路电流偏差 |
| 1002 | dVoc | 开路电压偏差 |
| 1003 | dImp | MPP 电流偏差 |
| 1004 | dVmp | MPP 电压偏差 |
| 1005 | dPmp | 最大功率偏差 |
| 1006 | dFF | 填充因子偏差 |
| 1007 | Rs | 串联电阻偏高 |
| 1008 | Rsh | 并联电阻偏低 |
| 1009 | step | IV 曲线台阶 |
| 1010 | Isc_near_zero | 短路电流接近零 |

## 根因异常 (2001-2021)

| 编码 | 根因 | 描述 |
|------|------|------|
| 2001 | 均匀遮挡/灰尘 | 均匀遮挡、灰尘覆盖导致光吸收减少 |
| 2002 | 组件老化 | 长期老化导致电池性能下降 |
| 2003 | 辐照测量偏差 | 辐照传感器故障或位置不当 |
| 2004 | 电流失配 | 组件间电流不一致 |
| 2005 | 旁路二极管故障 | 旁路二极管短路或异常导通 |
| 2006 | PID效应 | 电势诱导衰减形成分流通道 |
| 2007 | 温度测量偏差 | 温度补偿失真 |
| 2008 | 少接组件 | 组串实际组件数量少于设计值 |
| 2009 | 热斑效应 | 热斑区域形成高温与功率损失 |
| 2010 | 串联电阻增大 | 串联回路阻抗异常升高 |
| 2011 | 并联电阻下降 | 并联泄漏通道增强 |
| 2012 | 连接器接触不良 | 连接器氧化、松动或接触不稳 |
| 2013 | 线缆老化 | 线缆绝缘老化或铜芯氧化 |
| 2014 | 焊点退化 | 焊点疲劳退化导致导通恶化 |
| 2015 | 局部漏电 | 局部绝缘损伤形成漏电 |
| 2016 | 灰尘污渍导电 | 导电污染造成泄漏或绝缘问题 |
| 2017 | 局部遮挡 | 局部遮挡导致子串失配 |
| 2018 | 隐裂 | 隐裂引起组件局部电学隔离 |
| 2019 | 不均匀脏污 | 非均匀污染引起子串级失配 |
| 2020 | 玻璃碎裂 | 玻璃破损导致透光和安全风险 |
| 2021 | 组串断路 | 组串电流通路断开 |

## 运维建议编码 (3001-3021)

| 处理建议编码 | 对应根因编码 | 处理建议（一句话） |
|------:|------:|------|
| 3001 | 2001 | 首先确认检测是否在晴天下进行，以及组件是否存在大面积固定遮挡；现场无固定遮挡的情况下检查组串的积灰程度并在合适时进行清洗，干燥后进行复核检测。
| 3002 | 2002 | 检查现场组件并对组件的衰减程度进行评估，优先更换肉眼可见黄化且功率衰减对组串发电量造成显著影响的组件；对存在轻度或者中度衰减的组串定期进行IV检测以及发电性能追踪，在组件发生严重衰减前进行更换以避免风险。
| 3003 | 2003 | 检查现场辐照传感器安装的位置、倾角和朝向是否与组件一致，并对辐照传感器进行精度校准。
| 3004 | 2004 | 检查现场是否存在不均匀灰尘等脏污或者局部遮挡影响，不存在明显异常则使用EL或者红外成像仪对组件进行缺陷检测，若未发现上述问题则使用便携IV检测仪对每一片组件进行检测以定位问题组件。
| 3005 | 2005 | 检测组件接线盒、背板等位置是否存在异常痕迹，无明显异常则通过红外成像仪对组件进行温度检测，寻找是否存在大面积温度偏差组件，确认二极管短路或异常导通后优先更换接线盒或直接更换故障组件。
| 3006 | 2006 | 对疑似 PID 组串开展绝缘与衰减排查，如果电站大面积异常建议加装 PID 抑制、修复装置，以调整接地和进行反向偏压调节。
| 3007 | 2007 | 对气象站的温度传感器进行校准或更换，避免温度补偿持续失真。
| 3008 | 2008 | 核查组串实际接入组件型号、数量与系统配置是否一致，将系统配置修改为真实组件数量后进行复核检测。
| 3009 | 2009 | 通过EL或者红外成像仪定位存在热斑的组件，对于由于长期脏污、遮挡引起的热斑，在消除影响后定期进行IV检测进行问题追踪；对于组件内部故障导致的严重热斑进行组件更换。
| 3010 | 2010 | 首先沿组串电流路径排查线缆、接头、接线盒是否存在虚焊、老化、氧化等异常问题，若不存在明显异常则使用EL或者红外成像仪检测组件内部导通状态并定位内部故障，对引起串联电阻升高的问题部位或者组件进行紧固、修复或更换。
| 3011 | 2011 | 首先检查组件是否存在肉眼可见的背板破损，封边开裂、表面裂纹等问题，结合组件级的IV检测对并联电阻异常偏低的组件进行漏电与绝缘排查，确认组件外部或内部缺陷后进行组件更换。
| 3012 | 2012 | 现场关闭逆变器或断开直流侧开关后对 MC4 等连接器进行检查，对有问题的接头进行重新压接或更换，检查是否存在不同品牌或型号连接器混插使用的情况。
| 3013 | 2013 | 首先对现场的线缆进行目视检查，检查直流线缆、护套、MC 接头是否存在龟裂、硬化、开裂等问题，尤其是折弯处、穿墙、支架等摩擦点；若无明显外观异常，则根据现场情况对线缆进行绝缘检查或者热红外探测，对重点隐患进行直流线缆更换。
| 3014 | 2014 | 通过EL或者红外成像仪检查是否存在焊点处的异常高温或者焊点脱焊、互联断裂、暗区缺陷等故障，对高风险组件更换后进行复核检测。
| 3015 | 2015 | 首先检测现场组件是否存在由于组件表层、封边破损等问题导致的绝缘结构破坏问题，若不存在则通过红外成像仪识别存在异常发热漏电问题的组件，对存在严重风险的故障组件进行更换。
| 3016 | 2016 | 现场目视排查组件表面，检查是否存在大面积积污、斑块状污渍及导电类污染；对污染异常组件实施深度清洁处理，完成后开展复检校核。必要时开展湿态绝缘检测，验证表层漏电通路是否形成，确保绝缘性能恢复，消除泄漏隐患。
| 3017 | 2017 | 现场检查组串附件是否存在由于树影、异物、支架或周边设施造成的局部遮挡，及时对可以移动的遮挡物进行处理后复核检测；对于无法进行移动的固定物遮挡，通过红外成像仪检测是否已经产生由于遮挡导致的热斑等问题，对于存在风险的热斑进行定期的IV检测以跟进问题进展。
| 3018 | 2018 | 使用EL或者红外成像仪、无人机检测是否存在组件隐裂，对已经引起明显功率损失或严重热斑的组件进行及时更换。
| 3019 | 2019 | 检查是否存在因扬尘、雨水冲刷、顽固污渍淤积导致单片组件不均匀遮挡引发的组件内发电单元失配，针对不均匀脏污进行专项清洁，对处于严重污染区域的组串定期进行IV检测以及时发现问题。
| 3020 | 2020 | 检查组串下是否存在面板玻璃破碎的问题组件，若存在则尽快对组件进行更换并在完成后进行复核检测。
| 3021 | 2021 | 首先确认组串是否存在接入，若配置错误可以将当前组串设置为空接；若存在有效组串接入，关闭逆变器或断开直流侧开关并检查接头是否连接正常以及线路存在破损；接线以及线路正常的情况下检查组串开路电压是否正常以及逆变器内部是否存在断路。

## CSV 字段说明

`rulemap_final_summary.csv` 中目前使用以下数值字段：

- `rulemap_final_phenomena_codes`：分号分隔的表象编码列表
- `rulemap_primary_cause`：主根因编码
- `rulemap_final_causes`：分号分隔的根因编码列表
- `rulemap_primary_advice`：主运维建议编码
- `rulemap_final_advice_codes`：分号分隔的运维建议编码列表

对应的可读字段：

- `rulemap_final_phenomena`：表象文本，带数值
- `rulemap_final_phenomena_fields`：表象字段列表
- `rulemap_primary_cause_name`
- `rulemap_final_cause_names`
- `rulemap_primary_advice_text`
- `rulemap_final_advice_texts`
