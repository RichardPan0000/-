## 按标签整理数据

会生成这样的结构：

```text
<output_dir>/
  normal/
    task_id_inverter_id,string_id/
      plot.png
      metadata.json
  low_voc/
    task_id_inverter_id,string_id/
      plot.png
      metadata.json
```

其中：

- 一级目录是标签类别
- 二级目录名是 `task_id_inverter_id,string_id`
- `plot.png` 会画：
  - 当前组串
  - 同一个 `task_id + inverter_id + timestamp` 下其它组串
- `metadata.json` 保存这条样本的关键信息
