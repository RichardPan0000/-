# pytorch 之随机数





在 PyTorch 中，`torch.manual_seed(seed)` 是一个函数，用于设置生成随机数的种子。这个种子会影响到 PyTorch 中所有涉及随机数生成的操作，例如：

- 随机初始化神经网络的权重
- 使用随机采样生成数据
- 使用随机 dropout

**为什么要设置种子？**

在深度学习中，模型的训练和评估过程通常涉及大量的随机性。例如，神经网络的权重通常是随机初始化的，训练数据通常是随机打乱的，以及训练过程中可能会使用随机 dropout。

为了确保实验的可重复性，我们需要控制这些随机性。通过设置相同的种子，我们可以确保每次运行代码时，得到的随机数序列都是相同的。这样，我们就可以在不同的实验中进行公平的比较，并且可以重现我们的实验结果。

**`如何使用 torch.manual_seed()？`**

`torch.manual_seed()` 接受一个整数作为参数，这个整数就是随机数的种子。例如，要将种子设置为 42，可以这样做：

```
import torch

torch.manual_seed(42)
```

**设置种子后会发生什么？**

设置种子后，PyTorch 会将全局的随机数生成器的状态设置为一个特定的值。这意味着，每次你调用一个会生成随机数的操作时，你都会得到一个预先确定的值。

例如，下面的代码会生成两个相同的随机数张量：

```
import torch

torch.manual_seed(0)
x = torch.randn(2, 3)
print(x)

torch.manual_seed(0)
y = torch.randn(2, 3)
print(y)
```

**需要注意什么？**

- `torch.manual_seed()` 只会影响当前进程中的 PyTorch 随机数生成器。如果你使用多进程来加载数据或训练模型，你可能需要使用其他方法来设置种子，以确保所有进程都使用相同的种子。
- 除了 `torch.manual_seed()` 之外，还有其他一些函数可以用来设置种子，例如 `torch.cuda.manual_seed()`（用于设置 CUDA 设备的种子）和 `torch.random.seed()`（用于设置 Python 的随机数生成器的种子）。通常情况下，你只需要调用 `torch.manual_seed()` 就可以同时设置 CPU 和 GPU 的种子。





总而言之，`torch.manual_seed()` 是一个重要的工具，用于控制 PyTorch 中的随机性，以确保实验的可重复性。
