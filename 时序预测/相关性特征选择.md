





如果是做 **相关特征选择**，通常可以使用 **皮尔逊相关系数** 或 **Spearman 相关系数** 来筛选强相关或弱相关的特征，以减少冗余信息，提高模型的泛化能力。

------

### **方法 1：计算每个特征与目标变量的皮尔逊相关性**

使用 `pandas.DataFrame.corr()` 计算各个特征与目标变量的相关性：

```python
import pandas as pd

# 生成示例数据
data = {
    "feature1": [1, 2, 3, 4, 5, 6],
    "feature2": [2, 4, 6, 8, 10, 12],
    "feature3": [1, 3, 5, 7, 9, 11],
    "target": [1, 2, 3, 4, 5, 6]
}

df = pd.DataFrame(data)

# 计算特征与目标变量的相关系数
correlation_matrix = df.corr()

# 只取特征与目标变量的相关性（不包含特征之间的相关性）
target_corr = correlation_matrix["target"].drop("target")

print("特征与目标变量的相关性：\n", target_corr)
```

- 选择 **绝对值较大的特征**（通常大于 0.5）作为重要特征
- 过滤掉 **接近 0 的特征**，因为它们与目标变量无关

------

### **方法 2：使用 `scipy.stats.pearsonr` 计算相关性和 p 值**

如果你想计算 **显著性 p 值**，可以使用 `pearsonr`：

```python
from scipy.stats import pearsonr

for feature in ["feature1", "feature2", "feature3"]:
    corr, p_value = pearsonr(df[feature], df["target"])
    print(f"{feature} 与 target 的皮尔逊相关系数: {corr:.3f}, p 值: {p_value:.3f}")
```

- `p_value < 0.05` 说明相关性显著，可以考虑保留该特征

------

### **方法 3：剔除高度相关的特征**

如果两个特征 **彼此之间的相关性过高**，可以剔除其中一个，防止多重共线性：

```python
# 计算特征之间的相关性
feature_corr = df.drop(columns=["target"]).corr()

# 找出相关性大于 0.9 的特征对
high_corr_pairs = feature_corr[feature_corr.abs() > 0.9]
print(high_corr_pairs)
```

- 选择其中一个特征，删除另一个，以减少冗余

------

### **方法 4：Spearman 相关性（适用于非线性关系）**

如果数据存在 **非线性关系**，皮尔逊相关系数可能不适用，可以使用 Spearman 相关系数：

```python
from scipy.stats import spearmanr

for feature in ["feature1", "feature2", "feature3"]:
    corr, p_value = spearmanr(df[feature], df["target"])
    print(f"{feature} 与 target 的 Spearman 相关系数: {corr:.3f}, p 值: {p_value:.3f}")
```

- **Spearman 相关系数** 适用于 **排序关系**，例如时间序列、等级评分等数据

------

### **总结**

| 方法                               | 适用情况                   | 代码                                   |
| ---------------------------------- | -------------------------- | -------------------------------------- |
| **皮尔逊相关系数** (`pearsonr`)    | 线性关系，数据服从正态分布 | `df.corr()`                            |
| **Spearman相关系数** (`spearmanr`) | 非线性关系，数据排序相关   | `spearmanr(df[feature], df["target"])` |
| **过滤高相关特征**                 | 选择低共线性特征           | `df.drop(columns=["冗余特征"])`        |

**平稳性检测**（Stationarity Test）用于判断时间序列数据的统计特性（均值、方差、协方差）是否随时间变化。

在时间序列分析和预测中，**平稳序列** 更容易建模，而 **非平稳序列** 可能需要差分、变换等处理后才能用于建模。

------

## **1. 什么是平稳性？**

一个时间序列 XtX_t 是**平稳的**，如果它满足以下条件：

1. **均值平稳**（Mean Stationarity）：
   - 序列的均值不随时间变化，即 E[Xt]=μE[X_t] = \mu 为常数。
2. **方差平稳**（Variance Stationarity）：
   - 序列的方差不随时间变化，即 Var(Xt)=σ2\text{Var}(X_t) = \sigma^2 为常数。
3. **自协方差平稳**（Autocovariance Stationarity）：
   - 序列的自协方差 Cov(Xt,Xt+h)\text{Cov}(X_t, X_{t+h}) 只与时间间隔 hh 相关，而与具体时间点 tt 无关。

如果一个序列不满足上述条件，则称为 **非平稳序列**。

------

## **2. 为什么要进行平稳性检测？**

- 许多时间序列建模方法（如 **ARIMA、VAR、Granger因果分析**）**假设数据是平稳的**。
- **非平稳数据可能导致错误的模型估计**，甚至无法进行预测。
- 如果数据是**非平稳的**，通常需要进行 **差分（Differencing）** 或 **转换（如对数变换）** 使其变为平稳。

------

## **3. 如何检测平稳性？**

### **方法 1：绘制时序图**

直接绘制时间序列的折线图，观察趋势：

```python
import matplotlib.pyplot as plt
import pandas as pd

# 生成示例时间序列数据
time_series = pd.Series([1, 2, 4, 8, 16, 32, 64, 128, 256])

# 绘制时间序列
plt.plot(time_series)
plt.title("时间序列图")
plt.show()
```

如果图像**呈现明显的趋势或周期性**，说明可能是**非平稳**的。

------

### **方法 2：自相关图（ACF）**

- 如果时间序列是平稳的，**ACF（自相关函数）会快速衰减**。
- 如果时间序列是非平稳的，**ACF 会缓慢衰减**。

```python
from statsmodels.graphics.tsaplots import plot_acf

plot_acf(time_series)
plt.show()
```

- **ACF 迅速降为零 → 可能是平稳序列**
- **ACF 缓慢衰减 → 可能是非平稳序列**

------

### **方法 3：单位根检验（ADF检验）**

**Augmented Dickey-Fuller（ADF）检验** 是最常用的平稳性检验方法，检验序列是否存在 **单位根**（unit root）。

```python
from statsmodels.tsa.stattools import adfuller

result = adfuller(time_series)

print("ADF 检验统计量:", result[0])
print("p 值:", result[1])  # 关键指标
print("临界值:", result[4])  # 1%, 5%, 10% 置信区间的值

# 判断是否拒绝原假设（单位根存在，非平稳）
if result[1] < 0.05:
    print("拒绝原假设，数据是平稳的")
else:
    print("无法拒绝原假设，数据是非平稳的")
```

- **p 值 < 0.05** → 拒绝原假设，说明数据是平稳的 ✅
- **p 值 > 0.05** → 不能拒绝原假设，说明数据是**非平稳**的 ❌

------

## **4. 非平稳数据如何转换成平稳数据？**

如果数据是非平稳的，通常可以用以下方法转换：

1. **差分（Differencing）**

   - 用当前值减去前一个时间点的值，使数据稳定：

   ```python
   diff_series = time_series.diff().dropna()
   ```

   - 如果一次差分后仍然非平稳，可以进行**二阶差分**：

   ```python
   diff_series = time_series.diff().diff().dropna()
   ```

   - 如果数据是季节性的，可以用 **季节性差分**：

   ```python
   diff_series = time_series.diff(periods=12).dropna()
   ```

2. **对数变换（Log Transform）**

   - 适用于方差不稳定的数据：

   ```python
   import numpy as np
   log_series = np.log(time_series)
   ```

3. **Box-Cox 变换**

   - 适用于非正态分布数据：

   ```python
   from scipy.stats import boxcox
   boxcox_series, lambda_value = boxcox(time_series)
   ```

| 方法         | 目的             | 代码         |
| ------------ | ---------------- | ------------ |
| **时序图**   | 观察趋势、季节性 | `plt.plot()` |
| **ACF 图**   | 检测自相关性     | `plot_acf()` |
| **ADF 检验** | 检测单位根       | `adfuller()` |
| **差分**     | 消除趋势         | `.diff()`    |
| **对数变换** | 处理指数增长     | `np.log()`   |
