在时间序列分析中，**[ACF](https://zhida.zhihu.com/search?content_id=255720534&content_type=Article&match_order=1&q=ACF&zhida_source=entity)（[自相关函数](https://zhida.zhihu.com/search?content_id=255720534&content_type=Article&match_order=1&q=自相关函数&zhida_source=entity)，Autocorrelation Function）** 是用来度量一个序列与其滞后版本之间相关性的工具。周期性表现为在某个固定间隔（比如每隔12天、12个月）上自相关显著增强，因此我们可以通过 ACF 图（autocorrelation plot）来识别这种重复模式。

## 举个例子：产品销量

假设一个产品有明显的[季节性](https://zhida.zhihu.com/search?content_id=255720534&content_type=Article&match_order=1&q=季节性&zhida_source=entity)，比如每隔 7 天销量会上涨一次（周末销售增加）。我们模拟这样的数据并使用 ACF 来识别这个周期性。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.graphics.tsaplots import plot_acf

%config InlineBackend.figure_format = 'retina'  # 高清图像输出（Jupyter 环境）
plt.style.use('ggplot')                         # 使用 ggplot 风格


# 模拟销量数据（包含周期性）
np.random.seed(42)
days = 100
period = 7
noise = np.random.normal(0, 2, size=days)
sales = 10 + 5 * np.sin(2 * np.pi * np.arange(days) / period) + noise

# 转为 Pandas 时间序列
date_range = pd.date_range(start='2023-01-01', periods=days, freq='D')
sales_series = pd.Series(sales, index=date_range)

# 画出时间序列
plt.figure(figsize=(10, 4))
sales_series.plot(title='Product Sales Over Time')
plt.xlabel('Date')
plt.ylabel('Sales')
plt.grid(True)
plt.show()
```

我们来一起解读这张 **ACF（自相关）图**：

- **横轴（Lag）**：表示滞后期数，从 1 到 30；
- **纵轴（ACF 值）**：表示每个滞后期数下的自相关系数，范围 -1 到 1；
- **红色竖线**：代表每个滞后期的自相关系数；
- **红色阴影区域**：95% 的置信区间，如果竖线超出这区域，就说明该滞后期的自相关显著 ≠ 0。

### 解读这张图：

- **周期性迹象明显**：
  - 在 **lag = 7, 14, 21, 28** 附近，红色竖线显著高于置信区间，呈现周期性高峰。
  - 说明这个时间序列具有 **每 7 天重复一次的周期性变化**（比如周末销量高）。
- **非零滞后显著**：
  - lag=7、14、21、28 的值为正，说明当前销量与 7 天前、14 天前的数据有显著正相关。
  - lag=3、4、10 等处的竖线也超出了置信区间，但为负值，代表滞后这些天数时有反向相关（例如销量下降）。
- **周期规律清晰**：
- 这种规则的“正-负-正-负”交替出现在每隔约 7 天的滞后处，是典型的 **正弦型周期性信号的自相关特征**。

> 该 ACF 图表明销量数据中存在 **每 7 天重复一次的周期性模式**。你可以据此推测：销量数据可能与“每周一周期”的行为（如周末促销、用户购物习惯等）有关。

## 傅立叶变换

接下来，我们使用用 Fourier 变换或使用 seasonal_decompose 来分解趋势、季节性和残差。

- **`seasonal_decompose`：**适用于稳定的周期性时间序列（如每天或每月的频率）。
- **[快速傅里叶变换](https://zhida.zhihu.com/search?content_id=255720534&content_type=Article&match_order=1&q=快速傅里叶变换&zhida_source=entity)（FFT）**：用于识别隐藏的周期频率成分，特别适合复杂或不规则的周期。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.seasonal import seasonal_decompose
from scipy.fftpack import fft
from scipy.signal import periodogram

# 设置图像风格
%config InlineBackend.figure_format = 'retina'
plt.style.use('ggplot')

# -------------------------------
# 1. 模拟周期性销量数据
# -------------------------------
np.random.seed(42)
days = 100
period = 7
noise = np.random.normal(0, 2, size=days)
sales = 10 + 5 * np.sin(2 * np.pi * np.arange(days) / period) + noise

date_range = pd.date_range(start='2023-01-01', periods=days, freq='D')
sales_series = pd.Series(sales, index=date_range)

# -------------------------------
# 2. 使用 seasonal_decompose
# -------------------------------
decomposition = seasonal_decompose(sales_series, model='additive', period=7)

# 绘图
decomposition.plot()
plt.xticks(rotation=30)
plt.figure(figsize=(18, 8))
plt.suptitle('Seasonal Decomposition (Additive Model)', fontsize=14)
#plt.tight_layout()
plt.show()
```

- **Trend（趋势）**：显示销量随时间是否上升或下降。

- **Residual（残差）**：无法用趋势和季节性解释的部分。

## 附：FFT 数学原理解释

> **快速傅里叶变换（FFT）是用来将时域信号分解成不同频率的正弦波叠加，从而揭示出其中的周期性本质。**

### 1. **傅里叶变换（Fourier Transform）基本思想**：

任何周期性或有限长度的信号 x(t)，都可以分解成很多个不同频率的正弦波和余弦波的组合：



等价于复数形式的离散傅里叶变换（DFT）：



- x_n 是原始时间序列；
- X_k是频域下第 k 个频率成分的复数系数（包含幅度和相位）；
- N 是数据长度。

### 2. **FFT 是 DFT 的加速算法**：

DFT 的复杂度是 O(N^2)，而 FFT（如 Cooley-Tukey 算法）通过“分而治之”方法把复杂度降到：

O(NlogN)，这也被称之为快速傅立叶变换 FFT。这使得处理大规模数据的频率分析变得非常高效。

### 3. **频率、周期 与 FFT 的关系**

- FFT 得到的是频率分量的幅值（功率谱）；
- 每个频率对应的周期是：

周期频率

所以我们在周期图中画的是：

```text
plt.plot(1 / frequencies[1:], power[1:])
```

这样就能清晰看到周期性波动的强度。

## 案例中的作用

模拟的销量数据为：

```text
sales = 10 + 5 * sin(2πt / 7) + noise
```

FFT 通过分解发现主频是 1/7，所以周期是 7。这一点从 power 的最大值出现在周期=7 的地方可直接观察到。

## 全文小结

| 工具               | 原理                              | 优势                              |
| ------------------ | --------------------------------- | --------------------------------- |
| seasonal_decompose | 移动平均 + 拟合季节项             | 直观，适合规则周期（如周/月）     |
| FFT                | 正弦波分解 + 频域变换（数学稳固） | 快速，能发现任意周期性（整/非整） |
| ACF                | 滞后相关，展示重复模式            | 可视化明显，但对非整周期不敏感    |





``` python
"""
检查序列周期性
"""
import numpy as np
import pandas as pd
from scipy import signal
from scipy.fft import rfft, rfftfreq

try:
    from statsmodels.tsa.seasonal import STL
except Exception:
    STL = None

# ---------- 辅助函数 ----------
def robust_detrend(x):
    # 去线性趋势（简单）：减去线性拟合
    n = np.arange(len(x))
    mask = ~np.isnan(x)
    if mask.sum() < 3:
        return x - np.nanmean(x)
    p = np.polyfit(n[mask], x[mask], 1)
    return x - (p[0]*n + p[1])

def compute_periodogram_peak(x, fs=1.0, min_period=2, max_period=None, use_welch=True):
    """返回 (best_period_in_samples_or_None, peak_ratio, peak_power, freqs, power)"""
    x = np.asarray(x)
    n = len(x)
    if n < 8:
        return (None, 0.0, 0.0, None, None)
    # 填充/去NaN
    if np.isnan(x).any():
        x = pd.Series(x).interpolate(limit_direction='both').to_numpy()
    x = x - np.mean(x)
    if use_welch:
        freqs, power = signal.welch(x, fs=fs, nperseg=min(256, n))
    else:
        # FFT
        yf = np.abs(rfft(x))**2
        freqs = rfftfreq(n, d=1.0/fs)
        power = yf
    # 限定周期范围 -> 对应频率范围
    if max_period is None:
        max_period = n//2
    min_freq = 1.0 / max_period if max_period > 0 else 0
    max_freq = 1.0 / min_period
    sel = (freqs >= min_freq) & (freqs <= max_freq) & (freqs > 0)
    if not np.any(sel):
        return (None, 0.0, 0.0, freqs, power)
    freqs_sel = freqs[sel]
    power_sel = power[sel]
    # 找峰
    idx_peak = np.argmax(power_sel)
    peak_power = power_sel[idx_peak]
    peak_freq = freqs_sel[idx_peak]
    best_period = 1.0 / peak_freq if peak_freq>0 else None
    # 峰值比：最大峰 / median(其他频段)
    others = np.delete(power_sel, idx_peak)
    med = np.median(others) if others.size>0 else 1e-12
    peak_ratio = float(peak_power / (med + 1e-12))
    return (best_period, peak_ratio, peak_power, freqs, power)

def acf_peak_check(x, target_period, max_lag=None):
    """简单自相关：判断 target_period 附近是否有显著的自相关峰"""
    x = np.asarray(x)
    n = len(x)
    if target_period is None or n < 8:
        return False, 0.0
    # fill
    if np.isnan(x).any():
        x = pd.Series(x).interpolate(limit_direction='both').to_numpy()
    x = x - np.mean(x)
    # 通过 FFT 计算自相关
    corr = np.correlate(x, x, mode='full')[n-1:]
    corr = corr / corr[0]
    # 取附近窗口
    lag = int(round(target_period))
    if lag <= 0 or lag >= len(corr):
        return False, 0.0
    window = 2 if max_lag is None else max(1, int(max_lag//10))
    lo = max(1, lag-window)
    hi = min(len(corr)-1, lag+window)
    peak_val = np.max(corr[lo:hi+1])
    # 置信界：approx 1.96/sqrt(N)
    conf = 1.96 / np.sqrt(n)
    return (peak_val > conf), float(peak_val)

def seasonal_strength(x, period=None):
    """用 STL 或简单方差分解估计季节性强度（0..1）"""
    x = np.asarray(x)
    n = len(x)
    if n < 8 or period is None or period < 2:
        return 0.0
    if np.isnan(x).any():
        x = pd.Series(x).interpolate(limit_direction='both').to_numpy()
    try:
        if STL is None:
            raise ImportError
        stl = STL(x, period=int(round(period)), robust=True)
        res = stl.fit()
        seasonal = res.seasonal
        resid = res.resid
        # strength = 1 - var(resid) / var(resid + seasonal)
        num = np.var(resid)
        den = np.var(resid + seasonal)
        if den <= 1e-12:
            return 0.0
        strength = max(0.0, 1.0 - num/den)
        return float(strength)
    except Exception:
        # fallback: fold by period and compute variance across phase
        p = int(round(period))
        if p < 2 or p > n//2:
            return 0.0
        folded = np.array([x[i::p][:n//p] for i in range(p)])
        # 每相位的均值作为季节性分量
        phase_mean = np.nanmean(folded, axis=1)
        # 将相位均值展开回长度 n
        seasonal = np.tile(phase_mean, n//p + 1)[:n]
        resid = x - seasonal
        num = np.var(resid)
        den = np.var(resid + seasonal)
        if den <= 1e-12:
            return 0.0
        return float(max(0.0, 1.0 - num/den))

# ---------- 主函数：对单条序列判断 ----------
def detect_periodicity_single(x, fs=1.0, min_period=2, max_period=None,
                              spec_ratio_thresh=5.0, acf_conf_thresh=0.0, stl_strength_thresh=0.2):
    """
    返回 dict 包含:
      period (samples), spec_ratio, acf_ok(bool), acf_val, stl_strength, score
    """
    n = len(x)
    if n < 8:
        return dict(period=None, spec_ratio=0.0, acf_ok=False, acf_val=0.0,
                    stl_strength=0.0, score=0.0, decision=False)
    x_d = robust_detrend(x)
    period, spec_ratio, peak_power, freqs, power = compute_periodogram_peak(x_d, fs=fs,
                                                                            min_period=min_period,
                                                                            max_period=max_period,
                                                                            use_welch=True)
    acf_ok, acf_val = acf_peak_check(x_d, period)
    stl_s = seasonal_strength(x_d, period)
    # 各项得分（0..1 归一化）
    spec_score = min(1.0, np.log1p(spec_ratio)/np.log1p(100.0))  # 对数缩放
    acf_score = float(acf_val)  # 自相关本身通常在 0..1
    stl_score = stl_s  # already 0..1
    # 权重
    w_spec, w_acf, w_stl = 0.5, 0.3, 0.2
    score = w_spec*spec_score + w_acf*acf_score + w_stl*stl_score
    decision = (spec_ratio >= spec_ratio_thresh and stl_s >= stl_strength_thresh) or (score >= 0.4)
    return dict(period=period, spec_ratio=spec_ratio, acf_ok=acf_ok, acf_val=acf_val,
                stl_strength=stl_s, score=score, decision=decision)

# ---------- 批量处理 DataFrame（每列一条序列） ----------
def detect_periodicity_batch(df, fs=1.0, min_period=2, max_period=None,
                             spec_ratio_thresh=5.0, stl_strength_thresh=0.2):
    """
    df: pd.DataFrame, 每列为一条序列（index = time / 等间隔采样点）
    返回 DataFrame，列: period, spec_ratio, acf_ok, acf_val, stl_strength, score, decision
    """
    results = {}
    for col in df.columns:
        x = df[col].to_numpy()
        r = detect_periodicity_single(x, fs=fs, min_period=min_period, max_period=max_period,
                                      spec_ratio_thresh=spec_ratio_thresh,
                                      stl_strength_thresh=stl_strength_thresh)
        results[col] = r
    return pd.DataFrame(results).T

# ---------- 示例用法 ----------
if __name__ == "__main__":
    # 假设 df 每列为一条时间序列（等间隔采样）
    df = pd.read_csv("2024062616553_110B118S0257_data.csv", index_col=0)
    out = detect_periodicity_batch(df, fs=1.0, min_period=2, max_period=200,
                                   spec_ratio_thresh=5.0, stl_strength_thresh=0.2)
    print(out.sort_values("score", ascending=False).head(50))
    pass
```
