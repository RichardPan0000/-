## 2. 数据库配置

### 2.1 连接池

使用 `DBUtils.PooledDB` 管理连接池：

```python
class DataBase:
    def __init__(self):
        self.mysql_pool = PooledDB(
            creator=pymysql, maxconnections=10, mincached=1,
            blocking=True, ping=1, ...
        )
        self.starrocks_pool = PooledDB(
            creator=pymysql, maxconnections=10, mincached=1,
            blocking=True, ping=1, port=9030, ...
        )
```

### 2.2 关键查询

| 方法 | 数据源 | 用途 |
|------|--------|------|
| `get_ai_area()` | MySQL | 获取所有时区列表 |
| `get_ai_station(area)` | MySQL | 获取指定时区的站点ID列表 |
| `get_ai_station_with_tz(area)` | MySQL | 获取站点ID+working_mode+feature_a |
| `get_station_info(station_id)` | MySQL | 获取站点经纬度、时区等信息 |
| `get_station_his_data(...)` | StarRocks | 获取站点历史统计数据 |
| `get_station_load_his_filtered_data(...)` | StarRocks | 获取滤波后的负荷数据 |
| `get_station_list_load_his_filtered_data(...)` | StarRocks | 批量获取滤波负荷数据 |

### 4.2 模型加载流程

```python
def load_model_checkpoints(is_from_s3=True):
    if is_from_s3:
        # 从 S3 下载到本地
        s3_conn.download('sigen-load-prediction/dlinear/...', 'tsxer_main/checkpoints/...')
        s3_conn.download('sigen-load-prediction/timexer/...', 'tsxer_main/checkpoints/...')
    
    # 加载 DLinear 权重
    dlinear_cks = torch.load(check_path_dlinear, map_location='cpu')
    
    # 加载 TimeXer 权重（去除 DataParallel 前缀）
    timexer_cks = torch.load(check_path_timexer, map_location='cpu')
    if has_module_prefix(timexer_cks):
        timexer_cks = {k.replace("module.", ""): v for k, v in timexer_cks.items()}
    
    return dlinear_cks, timexer_cks
```

## 7. 线程模型

系统启动后的线程结构：

```
Main Thread (model_dispatch.py)
│
├── LoadWhiteList Thread (周六计算)
├── LoadWhiteListBadCase Thread (周日计算)
├── LoadModelTrain Thread (每日训练)
├── LoadModelPrediction Thread (每日预测)
├── LoadModelPredictionBackUp Thread (备份预测)
├── LoadModelPredictionHolidayOn Thread (节假日开启)
├── LoadModelPredictionHolidayOff Thread (节假日关闭)
├── LoadRecorder Thread (最大负荷记录)
├── LoadModelPredictionSupplementary Thread (补充预测)
└── LoadModelPredictionFilter Thread (过滤预测)
    │
    └── ThreadPoolExecutor (每个预测任务内部)
        ├── Worker 1: batch prediction
        ├── Worker 2: batch prediction
        └── ...
```
